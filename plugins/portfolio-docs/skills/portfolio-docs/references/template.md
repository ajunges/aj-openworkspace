# CAMADA 1 — SUMÁRIO UNIVERSAL

> **Propósito:** explicar o produto em 1 página para qualquer pessoa da organização (novo colaborador, RH, finanças, time de outro produto).
> **Obrigatória:** Sim. **Público:** Todos.
> **Regra de redação:** linguagem acessível, sem sigla não explicada, prosa dominante (não lista). Se não couber em 1 página impressa, está detalhado demais para esta camada.

## O que é
*2-3 frases explicando o produto em linguagem simples. Sem jargão técnico pesado.*

## Para quem é
*Descrição do cliente típico em linguagem humana. Não é o ICP formal — é uma pintura.*

## Dor que resolve
*O problema real do cliente, antes de falar da solução. Se essa frase não doer, a camada está fraca.*

## Como se diferencia
*Uma frase com a diferença central. Não é "nossa plataforma é completa e segura" — é uma afirmação contestável e defensável.*

## Como ganha dinheiro
*Modelo de receita simplificado. "Licença perpétua, pagamento único parcelado em 12x" é suficiente. Não precisa detalhar aqui.*

## Status atual
*Onde estamos: primeiros clientes, escala, mature, etc. 1-2 frases.*

*3 marcos com prazo. Se não tem marco definido, é sinal de que o produto não tem plano.*
